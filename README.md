# اسمارٹ تنخواہ سسٹم — PWA

یہ ایک آفلائن چلنے والی ویب ایپ (PWA) ہے جو eSSL روزانہ حاضری فائلوں سے تنخواہ کا حساب جوڑتی ہے۔
سارا ڈیٹا آپ کے اپنے آلے (موبائل/کمپیوٹر) میں محفوظ رہتا ہے — انٹرنیٹ پر کہیں نہیں جاتا۔

---

## اس فولڈر میں کیا کیا ہے (یہ ساری فائلیں GitHub پر اپلوڈ کرنی ہیں)

| فائل | کام |
|---|---|
| `index.html` | اصل ایپ (اردو) |
| `xlsx.full.min.js` | Excel پڑھنے/لکھنے کی لائبریری (مقامی، تاکہ آفلائن چلے) |
| `manifest.json` | ایپ کی معلومات (نام، آئیکن، رنگ) |
| `sw.js` | service worker (آفلائن سہولت) |
| `icon-192.png` | آئیکن |
| `icon-512.png` | آئیکن |
| `icon-512-maskable.png` | آئیکن (Android maskable) |

> **اہم:** ساری فائلیں ایک ہی جگہ (repository کی جڑ / root) میں رکھیں — کسی ذیلی فولڈر میں نہیں۔

---

## GitHub Pages پر ڈالنے کا طریقہ

1. **github.com** پر اکاؤنٹ بنائیں/لاگ اِن کریں۔
2. اوپر دائیں **＋ → New repository** دبائیں۔ نام رکھیں مثلاً `tankhwah`۔ **Public** رکھیں۔ **Create repository** دبائیں۔
3. نئے repo میں **Add file → Upload files** دبائیں، اور اس فولڈر کی **ساری 7 فائلیں** کھینچ کر ڈال دیں۔ نیچے **Commit changes** دبائیں۔
4. اوپر **Settings → Pages** کھولیں۔
5. **Build and deployment → Source** میں **Deploy from a branch** چنیں۔ نیچے **Branch: `main`** اور فولڈر **`/ (root)`** رکھ کر **Save** دبائیں۔
6. ایک دو منٹ بعد اوپر آپ کا لنک آ جائے گا:
   `https://<آپ کا یوزرنیم>.github.io/tankhwah/`

---

## موبائل/کمپیوٹر پر ایپ کی طرح انسٹال کرنا

- **Android (Chrome):** لنک کھولیں → مینو (⋮) → **Add to Home screen / Install app**۔
- **iPhone (Safari):** لنک کھولیں → Share (⬆️) → **Add to Home Screen**۔
- **کمپیوٹر (Chrome/Edge):** ایڈریس بار کے دائیں **Install** کا آئیکن دبائیں۔

انسٹال کے بعد ایپ آئیکن سے کھلے گی اور **انٹرنیٹ کے بغیر بھی** چلے گی (پہلی بار کھولنے کے لیے انٹرنیٹ ضروری ہے تاکہ فائلیں محفوظ ہو جائیں)۔

---

## ایپ کو اپڈیٹ کرنا

جب بھی `index.html` یا کوئی فائل بدلیں:
1. GitHub پر نئی فائل اپلوڈ کریں (پرانی کی جگہ)۔
2. `sw.js` میں سب سے اوپر `const CACHE = 'payroll-v1';` کے نمبر کو بڑھا دیں (مثلاً `payroll-v2`) — اس سے ایپ نیا نسخہ لے لے گی۔

---

## نوٹ

- آپ کی محفوظ سیٹنگ (اساتذہ، اوقات، دستی منٹ وغیرہ) اُسی آلے کے اُسی براؤزر میں رہتی ہے۔
- نیا آلہ یا براؤزر بدلنے پر: پرانے میں **"سیٹنگ بیک اپ (JSON)"** سے فائل نکالیں اور نئے میں **"سیٹنگ واپس لائیں"** سے ڈالیں۔
