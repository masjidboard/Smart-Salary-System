کیٹیگری کے اوقات (حفظ / دینیات / ڈیفالٹ وغیرہ)

• صبح = صبح کا مقررہ ٹائم (اس کے بعد لیٹ)۔ • صبح ختم / دوپہر ختم = سیشن کب ختم ہوتا ہے۔ • گھنٹے اِنہی شروع و اختتام سے خودکار نکلتے ہیں (چاہیں تو ہاتھ سے بھی بدلیں)۔
ہر کیٹیگری کا اپنا اختتام ہو سکتا ہے (مثلاً دینیات کا صبح ختم 10:50)۔ تاریخ خالی = ہمیشہ۔ کسی تاریخ سے بدلے تو نئی سطر + "تاریخ سے"۔ کسی خاص دن کا الگ ٹائم ہو تو "دن" منتخب کریں۔

خصوصی اوقات — کسی مخصوص ID کے لیے

یہ کیٹیگری کے اوقات پر فوقیت رکھتا ہے۔ صرف وہی خانہ بھریں جو بدلنا ہو؛ خالی خانہ کیٹیگری سے لیا جائے گا۔
• مستقل الگ ٹائم → تاریخ و دن خالی/"ہر دن"۔
• کسی خاص دن کا الگ ٹائم → "دن" منتخب کریں (مثلاً ID 27: ایک سطر "ہر دن 7:40"، دوسری سطر "جمعرات 7:30")۔ زیادہ مخصوص قاعدہ خودبخود غالب آتا ہے۔
• کسی ایک تاریخ کے لیے → "تاریخ سے" و "تاریخ تک" ایک ہی۔ • پورے ہفتے کے لیے → 7 دن کی رینج۔
